## ✨ 현재 지원 기능
* **사도별 질문 및 3점 답변 정리** 연회장 상호작용 선택지 중 호감도를 가장 높게 쳐주는 답변만 한눈에 보기 쉽게 정리했습니다.
* **PC, 모바일 전체 가능**: PC는 기본, 스마트폰에서도 한손으로 편하게 검색하고 확인할 수 있는 반응형 UI를 제공합니다.
---

## 🛠 사용된 기술
* **Frontend**: React.js, Vite
* **Data**: Google Sheets API (via PapaParse)
* **Deployment**: Vercel
* **Icons**: Lucide-React
